# MASTER PRODUCTION-READY PROMPT

## Project Name

**Enterprise Bissi / Committee / Chit Fund / Lottery & Loan Management System**

## Objective

Develop a **production-ready, enterprise-grade, scalable, secure, multi-tenant Bissi/Committee Management Software** that supports customer management, lottery management, token management, loan management, collections, branches, collectors, reports, notifications, AI automation, and existing data migration from Excel and Google Sheets.

The software should be suitable for businesses managing thousands of customers, multiple branches, collectors, and daily financial transactions.

Use clean architecture, modular development, secure authentication, and production coding standards.
